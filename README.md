# Dr. [Your Name] — Mental Health Awareness Website

A calm, trilingual (English / Hindi / Gujarati) public mental health awareness site. Static HTML/CSS/JS — no build step, no server required.

## Files Included
- `index.html` (Site structure)
- `styles.css` (Design, tokens, & colors)
- `script.js` (Interactivity, routing, & lang switches)
- `translations.js` (Site copy database)

## Quick Start
1. Edit `translations.js` to change your name and bio text.
2. Open `index.html` directly in any web browser to test locally!
